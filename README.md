## 🔍 Troubleshooting

### The game runs slow

- Lower render distance
- Close extra apps
- Check RAM settings
- Update your graphics driver
- Make sure you use the latest release

### The client will not start

- Run the file as administrator
- Re-extract the folder
- Check if Windows blocked the file
- Install Java if needed

### The screen stays black

- Switch display mode from fullscreen to windowed
- Update your GPU driver
- Remove custom shaders or heavy resource packs
- Restart the client

### The controls feel delayed

- Turn off V-Sync
- Lower graphics settings
- Use fullscreen
- Close recording software and overlays
